Kemajuan teknologi saat ini sagatlah pesat. Teknologi merupakan senjata utama umat modern untuk membantu kehidupan manusia.

Berbicara tentang teknologi, banyak topik yang menarik untuk dibahas lebih dalam. Salah satu pekerjaan yang berhubungan dengan teknologi yaitu **_Frontend Engineer_**. Salah satu tugas Frontend Engineer adalah untuk membuat tampilan lebih menarik.

Untuk memulai belajar bagaimana menjadi seorang Frontend Engineer, hal pertama yang wajib dipelajari adalah mengenal apa itu HTML, CSS, dan Javascript. Ketiga hal tersebut adalah komponen penting yang saling berkaitan untuk membangun sebuah tampilan.

Disini, saya akan membahas ketiga hal tersebut dengan bahasa yang sederhana.

## HTML

HTML, singkatan dari **_Hypertext Markup Language_**. HTML digunakan untuk menyusun kerangka dari website dan mengorganisir tujuan dari masing masing bagian. Dengan HTML, kita bisa membuat section, heading, paragraf, anchor, dll dengan menggunakan tag dan attribute yang tersedia.

```html title="index.html"
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Muhammad Gata</title>
  </head>
  <body>
    <h1>Muhammad Gata</h1>
  </body>
</html>
```

Tetapi perlu diingat, bahwa HTML bukanlah bahasa pemrograman, HTML merupakan bahasa markup untuk membentuk struktur website. Itu dikarenakan HTML tidak mempunyai logika didalamnya.

## CSS

CSS atau **_Cescading Style Sheets_** adalah pendukung dari HTML untuk membuat tampilan lebih menarik. CSS berfungsi untuk mengatur tata letak (layout), warna, font, dll. Dengan CSS, tampilan yang berupa kerangka HTML dapat kita beri style dengan sesuka hati.

```css title="style.css"
h1 {
  color: 'white';
  background-color: 'black';
  font-size: '16px'
  font-family: Arial, Helvetica, sans-serif;
}
```

CSS memiliki properti dan value yang memberikan kebebasan dalam memberi style tampilan berdasarkan kerangka HTML kita.

## JAVASCRIPT

Javascript adalah bahasa pemrograman yang populer untuk menunjang pekerjaan disisi frontend. Javascript bekerja dengan cara memanipulasi DOM (Document Object Model) pada HTML dan kita bisa memberikan logika sesuai keinginan. Dengan Javascript, kita akan membuat tampilan menjadi lebih interaktif

```js showLineNumbers title="script.js"
const h1Element = document.querySelector("h1");

h1Element.style.color = "black";
h1Element.style.backgroundColor = "white";
```

---

Itulah skill wajib yang harus dimiliki oleh seorang Frontend Engineer. Jika diibaratkan, HTML itu seperti kerangka manusia, CSS seperti pakaian manusia, dan Javascript seperti otak yang memiliki kemampuan untuk berlogika.
